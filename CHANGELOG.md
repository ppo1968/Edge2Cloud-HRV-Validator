# CHANGELOG

## v0.4.4_importador_universal

### Added
- Importador universal de datos RAW en CSV/Excel.
- Detección automática de columnas RR/IBI/NN.
- Conversión automática de segundos a milisegundos.
- Exportación de `RRSeries` normalizadas.
- Auditoría de importación RAW.

- No se calcula HRV hasta disponer de una serie RR normalizada y auditada.
