# Manual técnico

## Objetivo técnico

Estandarizar el procesamiento y comparación de variables HRV entre sistemas edge, cloud y sistemas de referencia.

## Componentes

- Lectura de datos.
- Limpieza y normalización.
- Cálculo de variables HRV.
- Comparación entre sistemas.
- Exportación de resultados.

## Variables principales

HR_bpm, RMSSD_ms, LnRMSSD, RR_medio_ms, SDNN_ms, SD1_ms, SD2_ms, indice_estres, frecuencia_respiratoria_resp_min, LF_ms2, HF_ms2, LF_nu_pct, HF_nu_pct, LF_HF_ratio.
