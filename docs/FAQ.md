# FAQ

## ¿Se suben los datos a GitHub?

No. Las carpetas `Datos/` y `Resultados/` deben permanecer en local.

## ¿Qué contiene GitHub?

Código, documentación, ejemplos, roadmap, changelog y materiales del proyecto.

## ¿Para qué sirve este proyecto?

Para validar y comparar métricas HRV entre procesamiento edge, cloud y referencias externas.
