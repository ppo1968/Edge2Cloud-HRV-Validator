# Manual de usuario

## Objetivo

Este programa permite comparar variables de HRV calculadas en entornos edge y cloud.

## Uso básico

1. Colocar los datos en la carpeta local `Datos/`.
2. Ejecutar el programa principal.
3. Revisar los resultados generados en `Resultados/`.

## Carpetas locales

Las carpetas `Datos/` y `Resultados/` no deben subirse a GitHub.
